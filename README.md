## INIAD MOOCs: Random Event (a版)
※ この拡張機能は非公認の物です。  
この拡張機能はINIAD MOOCsのトップ画面にランダムで発生するイベントを追加するものです。  
トップ画面を読み込むたびにランダムで選ばれたイベントが発生します。  
イベントはa版のみで9種類です。(ゲームのイベントをMOOCs内で再現したものがメイン)

## 導入方法
1. このリポジトリページの右側にあるreleaseをクリックし、一番上のRandomEvents-v0.1.0.zipをダウンロードします。  
2. 新しいフォルダーを適当な場所に作ってから、ダウンロードしたzipファイルをフォルダーの中に展開します。
3. chromeの右上の点が三つ縦に並んでいるボタンをクリックし、「拡張機能」の項目にカーソルを合わせて、「拡張機能の管理」をクリック。画面遷移後、「パッケージされていない拡張機能を読み込む」をクリックし、先ほど作成したフォルダーを読み込ませると、導入完了です。


## 注意
発生するイベントの中にはジャンプスケアを含むものがあります。  

イベントを再抽選するために、ページの読み込みを何回も繰り返すと大学のサーバーへのリクエストが増加し、サーバーに負荷をかける可能性があります。そのため、トップページの左上にクライアント側の動作のみでイベントを再抽選できるリロードボダンを用意したのでそれを活用してください。
